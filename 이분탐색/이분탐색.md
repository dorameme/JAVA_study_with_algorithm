정확한 이분탐색 정리      
상황	반복조건	mid 계산	구간 줄이기 (조건 참일 때)	구간 줄이기 (조건 거짓일 때)      
1. 특정값 찾기	L <= R	mid = (L + R) / 2		L = mid + 1, R = mid - 1     
2. 최대값 찾기	L < R	mid = (L + R + 1) / 2	L = mid (mid 포함)	R = mid - 1 (mid 제외)      
3. 최소값 찾기	L < R	mid = (L + R) / 2	    R = mid (mid 포함)	L = mid + 1 (mid 제외)
